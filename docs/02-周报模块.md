# 模块文档 · ②AI 周报(选题 TOP10 + 串烧初稿)

> 每周日把一周的 AI 早报拉通,产出两份可直接用的创作素材:周维度选题榜 + 口播串烧初稿。周报的价值 = 把单日散点拼成一周叙事线(例:"中美AI脱钩战"贯穿五天的主线,单日日报看不出来)。

## 每周日时间线(GST)

```
08:27  云端 routine(trig_01K7BNkmM49UM8KzHj7x31Vs)
       ① clone AI-Daily-Briefing(只读!)读 briefings/ 近7天早报(约14份)
       ② 可选补充: 本仓库 待审核/ 的本周日报
       ③ 按 reference/爆款选题评分框架.md(周维度热度=同一事件多天/多板块反复出现)
          → 周报/日期/本周选题TOP10.md(选题卡格式)
       ④ 按 reference/串烧写作规范.md(形态B)
          → 周报/日期/一周AI串烧初稿.md(8-12聚类锚点,3分钟内,≥30信息行)
       ⑤ 推 main
08:3x  weekly-push Action → 两份文档发 Hotspot 机器人
```

## 文件去向

| 文件 | 说明 |
|---|---|
| `周报/日期/本周选题TOP10.md` | 10 张选题卡:排名+爆款指数/钩子标题×2-3/切入角度/适配平台+写作skill/竞争度/时效/避坑 |
| `周报/日期/一周AI串烧初稿.md` | 口播串烧稿,【】聚类锚点可直接喂分镜 skill(koubo-fenjing-shipin) |
| `reference/串烧写作规范.md` | 形态B写作铁律(一行一信息点/无连接词/无解说腔/态度只在收尾),从口播 skill 提炼 |

## 铁律与边界

- **AI-Daily-Briefing 是另一条生产线**:本模块只允许 clone 只读,严禁向它写入/推送(曾经误存过去,已回迁,勿重蹈)
- 素材不足 3 份早报→跳过本周,不编造
- 串烧总时长控制在 3 分钟内(约 45-50 信息行封顶)——07-11 那期你手改过"3分钟控速版",已按此收紧规范

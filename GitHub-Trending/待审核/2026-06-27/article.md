# 今日 GitHub Trending：一个 AI 工具包，两个自托管项目

> 今天的 GitHub 日榜里，AI 项目多是近期出现过的老面孔，真正新上榜的 AI 工具只有 AWS 官方的智能体工具包；另外两个涨幅靠前的，是自托管和本地运行的工具。三个项目方向不同，放在一起看更像一份"开发者自己掌控工具链"的样本。

---

## 1. agent-toolkit-for-aws — AWS 官方维护的 AI 编码智能体工具包

| 总 Star | 24h 新增 | 主语言 | 协议 |
|---|---|---|---|
| 1.4k | 243 | Python | Apache-2.0 |

它要解决的问题是：让 Claude Code、Codex、Cursor、Kiro 这些 AI 编码工具在操作 AWS 时，有一套官方维护的工具、知识和约束，而不是各自拼凑社区 MCP 服务器。工具包把 AWS 的服务选择、部署、运维等能力打包成可以直接安装的插件。

与社区自建的 AWS MCP 服务器相比，差异在于它由 AWS 官方维护，并已标注为 GA（正式可用）。它按场景拆成几个插件：aws-core 覆盖服务选型、CDK/CloudFormation、Serverless、容器、存储、可观测性和计费；aws-agents 面向用 Amazon Bedrock 和 AgentCore 构建智能体；aws-data-analytics 覆盖 S3 Tables、Glue、Athena 的数据湖与 ETL；另有一个面向 DevSecOps 的插件用于排查事件、审计代码和做漏洞扫描。这些插件已上架 Anthropic 官方插件市场（claude-plugins-official），也能在 Codex、Cursor、Kiro 里以各自的方式接入。

适合已经在用 AI 编码工具、且主要技术栈在 AWS 上的团队。仓库创建于 2026 年 4 月，目前 1.4k star，处于早期阶段，但有官方背书。

🔗 项目地址：https://github.com/aws/agent-toolkit-for-aws

## 2. TREK — 自托管的协作式旅行规划工具

| 总 Star | 24h 新增 | 主语言 | 协议 |
|---|---|---|---|
| 7.7k | 1.1k | TypeScript | AGPL-3.0 |

TREK 是一个可以自己部署的旅行规划工具，主打"行程数据放在自己服务器上"。它提供行程规划、地图、预算与费用分摊、打包清单、旅行日志等功能，并支持多人实时协作和 PWA（可装到手机桌面）。

和 Wanderlog 这类商业旅行规划应用相比，它的差异主要在"自托管"和数据自主权——通过 Docker 部署后，行程、地图标注、开销记录都留在用户自己的实例里。项目还提供 SSO 登录、已访问国家的地图册等功能；README 提到内置了 AI 辅助，但产品定位仍以行程规划为主。

适合注重数据隐私、愿意自建服务的个人或小团队，以及想替代商业行程 App 的用户。项目创建于 2026 年 3 月，目前 7.7k star、652 fork，并提供在线 demo（demo.liketrek.com）。

🔗 项目地址：https://github.com/mauriceboe/TREK

## 3. MediaCrawler — 多平台公开内容采集工具

| 总 Star | 24h 新增 | 主语言 | 协议 |
|---|---|---|---|
| 53k | 673 | Python | 自定义 |

MediaCrawler 是一个面向中文自媒体平台的公开数据采集工具，覆盖小红书、抖音、快手、B 站、微博、贴吧、知乎等平台，支持关键词搜索、指定帖子与创作者主页抓取、二级评论采集，以及评论词云生成。

技术上，它基于 Playwright 浏览器自动化，靠保留登录态的浏览器上下文配合 JS 表达式获取签名参数，从而绕开对加密算法的逆向。开发者称这种做法降低了上手门槛——不必逆向各平台复杂的签名逻辑。仓库自带 IP 代理池、登录态缓存等工程能力，另有一个 Pro 版本提供断点续爬、多账号等特性。

仓库创建于 2023 年 6 月，目前 53k star、近 1.1 万 fork，是这一方向里有规模的项目之一。需要说明的是，作者在 README 顶部明确声明仅供学习研究、禁止商业用途，并提示爬虫存在合规风险，使用前需自行评估。

🔗 项目地址：https://github.com/NanmiCoder/MediaCrawler

## 今日观察

把今天这三个项目放在一起，看到的不是同一个主题，而是两条线：一条是 AI 编码工具链开始"官方化"，云厂商亲自下场维护插件；另一条是自托管、数据自主权类项目持续有人关注。AI 热点本身今天反而偏冷，日榜上多为近期已经出现过的旧项目。
